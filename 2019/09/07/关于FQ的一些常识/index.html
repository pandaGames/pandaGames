2019-09-07

翻墙问答：公安如何得悉你用翻墙软件?

李：公安未必能知道你透过翻墙软件在浏览什么，但公安要知道你在使用翻墙软件就绝对不难。过往我们亦收到不少听众报告，指他们在大学内使用翻墙软件或VPN，因而被大学校内警告或劝谕停止使用翻墙软件。

因为自由门这类翻墙软件，实际上连接代理主机，除非自由门设立十分大量主机群，以提供大量不同的IP连接代理主机，否则中国政府很容易知道那些IP实际上是自由门主机，就算未必敢屏蔽自由的代理主机服务都好，都会知道有那些主机正连接自由门之类的翻墙软件。因此，中国当局可以经常对在办公室，或大学校园使用自由门的人作出书面警告。

除了自由门之类的软件，就算使用VPN之类技术来翻墙都有同样的问题，因为VPN都需要经过固定IP的主机上网，中国当局有可能对大量中国IP连接的海外VPN提高警觉。因此，无论自由门之类的翻墙软件，还是VPN，要更进一步避过中国当局监视，就可能需要新技术，能全中国翻墙的网络交通，并不集中于个别IP，而是分散到数以百计，以至千计的IP。但暂时而言，这样做有相当的难度，一方面需要更多的经费，去租用相当大量的IP。另一方面，由于现时IPv4的网络地址已经几乎耗尽，很难腾出新IP作翻墙用途。因此，未来翻墙软件或技术如果能向IPv6技术过渡，由于IPv6可用的网址数目极大，那就可以分散到数以千计甚至万计的IP来避免中国当局的监控。

DC：最近YahooVoice的VoIP服务，有四十多万未经加密的用户名和密码被黑客偷走并外泄，请问这次意外，会否直接影响到雅虎中国用户的安全？

李：对大部分中国用户不会受直接影响，因为只有你曾经登记使用YahooVoice服务的雅虎用户才会受到这次事件影响。只不过，如果你在海外使用过雅虎中国的帐户，登记使用YahooVoice服务的话，那请你立即改变密码，并且密切留意阁下的雅虎电邮帐户以及信用卡有否有异动，因为有可能黑客会入侵你的电邮户口，甚至偷用你的信用卡。

只不过，这件事反映雅虎的用户资料库或个别主机的设计，有可能出现保安漏洞，令黑客可以透过一种叫SQLInjection的入侵手法，偷取个别服务资料库内与用户名称和密码相关敏感资料。这件事亦反映，雅虎并非在全部服务上，对用户资料作出加密。当雅虎用户资料库都有这种隐患时，就算你并非使用雅虎中国服务，这亦不排除中国的黑客，特别受聘于当局的黑客，尝试利用SQLInjection的漏洞，试图偷取中国用户在海外雅虎存有的密码资料。因此，就算使用雅虎中国以外的海外雅虎服务，仍然有需要对自已的密码和个人资料提高警觉。

如何防止查水表
今 天睡完午觉起来，刚到办公室，接到一个电话。那头一个深沉的男音：“请问你是XXX吗？”我：“是的。”男音：“这里是XXX市公安局网监大 队。。。”我当时心里想，这是哪个狗日的跟我在开玩笑呢。。。看看号码，中间带110数字，心想难道前段时间没翻墙发新时代被发现了，还是我微博违规被查 了。。。男音接着说：“你是不是做了一个个人网站。这个必须到网监大队来备案，请带上身份证以及家庭宽带安装回执，尽快到大队来办下相关手续。”我舒了一 口气，问了是不是必须备案，答曰：必须。于是我就去了。

到了公安局网监大队，看看是熟人，发根烟，填了下资料，结束后和他聊天，问他怎么知道我手机号码的，他说想知道一个人信息还不简单么，分分钟搞定，但是前 提得有一个人的名字。。。我想这不废话么。我又问，要是我在家上成人网会被查到么，他说可以查到，但是谁没事做这苦力，上成人网站的太多了，一个个查非得 累死，只会查典型。

接着又问，用代理软件翻墙上网会被爆菊么。。。他说，只要想查你，也是很容易的，就是工作量大了点，但是想查能查到。

然后楼主又发了根烟，猥琐的问，怎样才能不被你们查到呢。。。他眼睛一瞪：非法上网是不允许的！！！赶忙拿起打火机点上。他吸了口烟，幽幽的解释起来，哪种上网方式他们最讨厌，因为爆不了菊。。。

首先，确保你电脑里没有国产软件（明确的已知腾讯、360等会搜集你的信息，搜狗输入，迅雷之类都会有IP日志。不明确的可能搜集信息的还有很多）及其遗 留线程、模块 。有了这些国产软件，每个人在网上根本就是透明的。还有，国内各大社区论坛是时刻被监控的，包括天涯之类，每个大型论坛都会被网监部门要求主动提供一个有 禁号，封号，查看IP，禁IP这些权限的账号。各级公安系统均配备了舆情控制系统，能采集几乎所有的国内网站的发帖信息，检测到关键字就被单独列出。这就 是为什么国内网上一有敏感信息，总能在第一时间被删掉的原因所在了。

下面用最通俗的文字解释被网警查水表的过程，由网络整理而来，大家姑且看看。其实查不查水表在于自己，自己不作死，网警也不是闲的无聊，毕竟水表能查，但 是工作量太大，枪打出头鸟，像我们看看爱情动作片撸撸管的屌丝，网警们谁有这闲工夫来查你。。。说不定他们自己还逛大草榴呢。。。

1.不用代理

网警追踪流程：

网监　→　服务器IP　→　发帖人IP　→　发帖人ISP　→　档案

爆菊几率：100%

2.只用1层私人VPN

网警追踪流程：

网监　→　服务器IP　→　VPN的IP　→　路由日志　→　链接者IP　→　链接者ISP　→　办网档案

爆菊几率：100%

3.用1层共用VPN

网警追踪流程：

网监　→　服务器IP　→　VPN的IP　→　入侵VPN服务器　→　日志　→　链接者IP　→　链接者ISP　→　办网档案

爆菊几率：80%

4.用2层私人VPN

网警追踪流程：

网监　→　服务器IP　→　VPN2的IP　→　入侵VPN2服务器　→　日志查VPN1　→　路由日志　→　链接者IP　→　链接者ISP　→　办网档案

爆菊几率：40%

5.用2层共用VPN

网警追踪流程：

网监　→　服务器IP　→　VPN2的IP　→　入侵VPN2服务器　→　日志查VPN1　→　日志　→　链接者IP　→　链接者ISP　→　办网档案

爆菊几率：25%

6.用N层VPN，但是如果发帖人留有QQ

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　发帖人常用用户名　→　谷歌（百度）一下　→　查到QQ　→　找企鹅公司查近期登录IP

爆菊几率：100%

7.用N层VPN，如果发帖人无QQ

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　发帖人常用用户名　→　谷歌（百度）一下　→　查到其他论坛注册的ID　→　找管理员查注册（登录）IP

爆菊几率：100%

8.用N层私人（共用）VPN后删掉VPN，如果发帖人在网上没有任何信息

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　无常用用户名　→　根据发帖内容（如北京市海淀区XX小区）　→　监控整个小区宽带　→　分析　→　找到IP

爆菊几率：80%（如果不是严重的问题，如判国之类的应该不会用这种蛋疼方法）

9.用N层私人（共用）VPN后删掉VPN，如果发帖人在网上没有任何信息，但电脑上有自动链接网络的软件，如QQ，金山快盘等。。

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　无常用用户名　→　各大公司查IP　→　查到常用用户名　→　查历史登录记录　→　查ip　→　爆菊

爆菊几率：100%

10.到KFC之类的地方上公共网。

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　看发帖日期时间　→　到KFC　→　调监控　→　爆菊

爆菊几率：100%

11.到KFC之类的地方上公共网，假设本次没被监控拍到

网警追踪流程：

网监　→　服务器IP　→　访问网站　→　看发帖日期时间　→　到KFC　→　查MAC地址　→　到附近其他公共网络查此MAC　→　(如果DHCP会查计算机名)　→　附近监控　→　爆菊

爆菊几率：100%

12.假设网监已知道MAC地址

1 假设自己买的电脑

查MAC数据库能找到销售商。。一般买电脑的时候MAC也有记录。。可能是三包凭证之类的都有。。而一般买电脑会让你提供个人信息填写三包，MAC之类的 硬件代码也会记录。。之后你懂的，不过有些商家对这个不怎么记录。不过也有可能查购买者银行卡（如果非现金）。所以..爆菊几率<25%

2 假设是线下二手交易

同样，查到电脑原主人，问问就知道了。。不过一般交易者都会留下你的手机号之类的，到移动联通电信之类的查查就知道了。。如果非实名卡查查基站，能查到IMEI。 爆菊几率<75%

综上所述，最保险的方法就是用两层公用VPN。但是网监不是机器人，查水表也不是那么容易的，得花很多精力，除非大家傻的把自己家地址什么的公布在网上，一般没人有这个闲工夫来查你的。。。

所以大家能挂代理就挂吧，新时代最好还是挂代理发，自己的图床最好用不需要账号密码的。必须用账号登录才能上传的，不要设置和自己常用网名一样的账号.
